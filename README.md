# ai_prompt

AI에게 더 나은 결과를 얻기 위한 질문 프롬프트 템플릿을 저장한 저장소입니다.

- `prompt-templates/_INDEX.md`: 키워드 기반 템플릿 라우터
- `prompt-templates/`: 실제 프롬프트 템플릿 파일 모음
  - `travel_overnight.md`, `travel_daytrip.md`, `food_nearby.md`, `dev_newproject.md`, `dev_codeReview.md`, `stock_analysis.md`
  - `_CHANGELOG.md`: 템플릿 변경 이력
