# 질문 프롬프트 템플릿

AI에게 더 좋은 답변을 받기 위한 질문 템플릿입니다. 아래 구조를 따라 질문을 만들면 맥락, 목적, 조건을 명확하게 전달할 수 있습니다.

## 1. 목적(Goal)
- 내가 원하는 결과는 무엇인가요?
- 어떤 문제를 해결하려고 하나요?

예: "이 코드의 성능 문제를 찾아서 개선 방안을 알려줘"

## 2. 배경(Background)
- 상황 설명
- 관련된 정보나 조건
- 이미 시도한 방법

예: "파이썬 웹 스크래핑 코드인데, 1만 개 이상의 항목을 처리할 때 속도가 느립니다. requests와 BeautifulSoup를 사용합니다."

## 3. 세부 요구 사항(Requirements)
- 어떤 형식으로 답변을 받고 싶은가요?
- 제약 조건은 무엇인가요?
- 우선순위가 있는 항목이 있나요?

예:
- "코드 예시를 포함해서"
- "시간 복잡도가 낮은 방식으로"
- "파이썬 3.11 기준으로"

## 4. 질문(Question)
- 구체적이고 명확하게 묻기

예: "이 코드의 반복문을 최적화할 수 있는 방법을 구체적으로 설명해줘."

---

## 프롬프트 템플릿 예시

```
목적: 파이썬 웹 스크래핑 코드의 성능을 개선하고 싶습니다.
배경: 현재 requests와 BeautifulSoup를 사용해 1만 개 이상의 항목을 처리하는데 속도가 매우 느립니다. 네트워크 요청과 파싱 단계가 병목인 것 같습니다.
요구 사항:
- 코드 개선 방법을 단계별로 설명해주세요.
- 가능한 경우 코드 예시를 제공해주세요.
- Python 3.11 환경을 기준으로 해주세요.
질문: 어떤 방식으로 이 코드를 최적화할 수 있나요?
```

---

## 사용 팁
- 필요한 경우 예시 데이터나 현재 코드를 함께 제공하세요.
- "왜"와 "어떻게"를 모두 포함하면 더 상세한 답변을 얻기 쉽습니다.
- 요구 사항을 나열할 때 번호나 핵심어를 사용하면 AI가 빠르게 파악합니다.
